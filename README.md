# 📊 Loan Approval Analysis Dashboard (Power BI)

## 🔍 Project Overview
This project focuses on analyzing loan approval data to identify key factors influencing approval and rejection decisions. The dashboard provides interactive insights into applicant profiles, financial status, and risk categories.

## 🎯 Objectives
- Analyze loan approval trends
- Identify high-risk applicants
- Understand impact of income, credit history, and property area
- Improve decision-making using data visualization

## 🛠 Tools & Technologies
- Power BI
- Excel / CSV Dataset
- Data Cleaning & Transformation
- Data Visualization

## 📊 Key Features
- ✔ Loan Status Distribution (Approved vs Rejected)
- ✔ Income vs Loan Amount Analysis
- ✔ Risk Category Segmentation (Low / Medium / High)
- ✔ Credit History Impact on Loan Approval
- ✔ Interactive Filters (Slicers)

## 📈 Insights
- Applicants with good credit history have higher approval chances
- High loan-to-income ratio leads to higher rejection
- Urban applicants show higher loan approval rates
